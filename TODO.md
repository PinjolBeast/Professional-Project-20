## 🎨 Design System

### Color Palette
- Primary: Soft Pink (#F8D7DA → #F4C2C2)
- Secondary: Cream (#FFFDD0)
- Accent: Lavender (#E6E6FA)
- Accent 2: Peach (#FFDAB9)
- Accent 3: Sage Green (#BCE3C6)
- Text: Dark Gray (#4A4A4A)
- Background: Off-white (#FFF9F5)

### Typography
- Headings: "Playfair Display" (elegant serif) or "Great Vibes" (script)
- Body: "Poppins" or "Lato" (clean sans-serif)

### Animations
- Fade-in on scroll
- Slide-up for cards
- Hover scale effects
- Floating floral elements
- Smooth page transitions

---

## 🔧 Technical Stack
- HTML5, CSS3, JavaScript (Vanilla)
- Firebase Authentication (Email/Password)
- Web3Forms API (Contact form)
- AOS Library (Scroll animations)
- Font Awesome (Icons)
- Google Fonts (Typography)
